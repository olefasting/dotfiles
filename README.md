# dotfiles

This contains dotfiles for configuring new environments and most commonly used tools

## usage

Install by executing [install.sh](/install.sh) from the repository root directory

The following variables can be used to influence the install process:

- `IS_SERVER` can be set to 'true' or 1 to perform a server install. This means that workstation exclusive stuff will not be installed
